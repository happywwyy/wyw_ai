## shrink 收缩计算
- 溢出空间 300px 让子元素来shrink 分摊
- 分摊的比例？
  flex-shrink：2 ：1   500 ：400
  500 - 300 * 500 需要分摊的比例
  400 - 300 * 400 需要分摊的比例
- 分摊的权重
  总权重？ 500 * 2 + 400 * 1 = 1400 
  1000 / 1400 = 0.7142857142857143
  400 / 1400 = 0.2857142857142857

  300 * 0.714 = 214
  300 * 0.286 = 85.6

  left 500 - 214 = 285.6
  right 400 - 85.6 = 314.4
